Política de Privacidade

Última atualização: 15 de maio de 2026

1. Quem somos

A Easy Care é uma plataforma de tecnologia em saúde que oferece fichas médicas digitais acessíveis via QR Code para emergências. Nosso site é easycareapp.com.br e nosso e-mail de contato é contato@easycareapp.com.br.

2. Dados que coletamos

Coletamos os seguintes dados pessoais e de saúde:

  • Dados de identificação: nome, CPF, RG, data de nascimento, sexo
  • Dados de contato: telefone, e-mail, endereço
  • Dados de saúde (sensíveis): tipo sanguíneo, alergias, medicamentos, condições médicas, cirurgias
  • Dados de emergência: contato de emergência, convênio
  • Foto de perfil (opcional)

3. Finalidade do tratamento

Os dados são coletados e tratados exclusivamente para:

  • Criação e manutenção da ficha médica de emergência
  • Geração e disponibilização do QR Code de emergência
  • Acesso por profissionais de saúde em situações de emergência
  • Comunicação por e-mail relacionada à conta do usuário

4. Base legal (LGPD — Lei 13.709/2018)

O tratamento dos dados é realizado com base no consentimento expresso do titular (Art. 7º, I) e na tutela da saúde (Art. 11, II, f), especialmente para dados sensíveis de saúde.

5. Compartilhamento de dados

Os dados são compartilhados somente com profissionais de saúde que acessam o QR Code em situações de emergência. Não vendemos, alugamos ou compartilhamos dados com terceiros para fins comerciais.

6. Armazenamento e segurança

Os dados são armazenados em servidores seguros da Supabase (região São Paulo, Brasil), com criptografia em trânsito (TLS) e em repouso. Adotamos medidas técnicas e organizacionais para proteger seus dados contra acesso não autorizado.

7. Direitos do titular

Conforme a LGPD, você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados a qualquer momento
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão dos seus dados
  • Revogar o consentimento a qualquer momento
  • Portabilidade dos dados para outro serviço

Para exercer seus direitos, entre em contato: contato@easycareapp.com.br

8. Encarregado de Dados (DPO)

Nosso Encarregado de Proteção de Dados pode ser contatado pelo e-mail: contato@easycareapp.com.br

9. Retenção de dados

Os dados são mantidos enquanto o usuário possuir uma conta ativa. Após solicitação de exclusão, os dados são removidos em até 30 dias.

10. Contato

Em caso de dúvidas sobre esta política, entre em contato:
contato@easycareapp.com.br

Termos de Uso

Última atualização: 15 de maio de 2026

1. Aceitação dos termos

Ao utilizar a plataforma Easy Care, você concorda com estes Termos de Uso. Se não concordar com qualquer parte destes termos, não utilize o serviço.

2. Descrição do serviço

A Easy Care oferece uma plataforma para criação de fichas médicas digitais acessíveis via QR Code. O serviço é destinado a pessoas físicas que desejam disponibilizar seus dados médicos para acesso em situações de emergência.

3. Responsabilidades do usuário

  • Fornecer informações verdadeiras e atualizadas
  • Manter suas credenciais de acesso em segurança
  • Atualizar seus dados sempre que houver mudanças relevantes
  • Não utilizar o serviço para fins ilícitos

4. Limitação de responsabilidade

A Easy Care não se responsabiliza por decisões médicas tomadas com base nas informações da ficha. O serviço é uma ferramenta de auxílio à comunicação em emergências e não substitui avaliação médica profissional.

5. Pagamento

O serviço é oferecido mediante pagamento único de R$ 29,00 por ficha cadastrada, com acesso vitalício. Não há cobranças recorrentes.

6. Cancelamento e exclusão

O usuário pode solicitar a exclusão de sua ficha a qualquer momento pelo e-mail contato@easycareapp.com.br. Após exclusão, o QR Code deixará de funcionar.

7. Propriedade intelectual

Todos os direitos sobre a plataforma Easy Care, incluindo marca, design e código, são de propriedade exclusiva da Easy Care.

8. Foro

Fica eleito o foro da comarca de São Paulo/SP para dirimir quaisquer controvérsias decorrentes destes termos.
